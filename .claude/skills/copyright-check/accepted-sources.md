# Accepted figure/data sources (single source of truth)

The `/copyright-check` skill and the `copyright-critic` agent read this file
before judging provenance. A source listed here is **cleared for this
project** — do NOT flag a credit just because it names one of these.

## Public-domain / open (always fine)

- **FRED / St. Louis Fed** — US-government public-domain series. Cite the
  series IDs; link `fred.stlouisfed.org`. (FRED also redistributes some
  third-party index series, e.g. ICE BofA OAS `BAMLH0A0HYM2`, Moody's
  seasoned yields `AAA`/`BAA`. Crediting **FRED** for a FRED-hosted series is
  correct and sanctioned — do not flag it as "Moody's/ICE".)
- **SEC EDGAR** — public company filings (10-K, etc.). Link `edgar.sec.gov`.
- **US government statute / law** — e.g. Internal Revenue Code statutory rates.
  Public record, not copyrightable.
- **Wikimedia Commons** — per-image license; confirm the specific file's terms.
- **Author's own original work** — figures/diagrams created for this site.

## Free for education with attribution (fine; keep attribution + creditHref)

- **NYU Stern / Aswath Damodaran data library** (`pages.stern.nyu.edu/~adamodar`).
- **Ken French Data Library**, Tuck School of Business (Fama-French factors).

## Owner-confirmed acceptable for this project (do NOT re-flag) — 2026-05-31

The project owner has confirmed these are properly licensed / cleared for use

> Note: this is a record of the owner's clearance decision, not a legal

## Never allowed (the gate + critic should still flag these)

Textbook scans/figures (McGraw-Hill, Pearson, Cengage, Wiley, ...), Getty /
Shutterstock / other paid stock imagery, screenshots of paid charts, external
image hotlinks, and anything from the gitignored `materials/` folder.
